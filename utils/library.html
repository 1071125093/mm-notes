# 库收集

收集个人使用过或遇到的类库

# 工具库

  • lodash (opens new window)是一个一致性、模块化、高性能的JavaScript 实用工具库
  • ramda (opens new window)专为函数式编程而设计的工具库
  • qs (opens new window)带有一些附加安全性的请求参数 parse 和 stringify 库

# 时间处理

  • moment (opens new window)JavaScript 日期处理类库
  • dayjs (opens new window)一个轻量的处理时间和日期的 JavaScript 库，和 Moment.js 的 API 设计保持完全一样
  • date-fns (opens new window)轻量级的 JavaScript 日期库，纯函数实现支持模块化

# 请求处理

  • axios (opens new window)是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中

# 精度处理

  • big.js (opens new window)用于任意精度的十进制算术运算

# 本地存储

  • localForage (opens new window)一个快速、简单的 JavaScript 存储库(中文文档 (opens new window))

# 滚动处理

  • better-scroll (opens new window)重点解决移动端（已支持 PC）各种滚动场景需求的插件

# 事件处理

  • hotkeys-js (opens new window)用于捕获键盘输入和输入的组合键

# 文件处理

  • FileSaver.js (opens new window)保存/下载文件(受 blob 的大小和内存限制)
  • StreamSaver.js (opens new window)保存/下载文件(不受 blob 的大小和内存限制)
  • sheetjs (opens new window)用于 Excel 的读取和导出

# 富文本

  • braft-editor (opens new window) React 富文本编辑器

# 用户体验

  • driver.js (opens new window) 用户引导动画库

# Webpack

  • webpack-chain (opens new window)使用链式 API 来生成和简化 webpack 的配置的修改
  • speed-measure-webpack-plugin (opens new window)统计 webpack 在各阶段的构建速度
  • webpack-bundle-analyzer (opens new window)通过可视化分析 webpack 打包文件的大小

# CLI 相关

  • Inquirer.js (opens new window) 交互式命令行工具
  • enquirer (opens new window) 交互式命令行工具（更好看）
  • chalk (opens new window) 修改控制台内容输出的样式
  • shelljs (opens new window) 调用系统命令
  • commander.js (opens new window) 编写指令和处理命令行
  • yargs (opens new window) 命令行参数解析

# 文档

  • VuePress (opens new window) Vue 驱动的静态网站生成器
  • dumi (opens new window) 基于 Umi 为组件开发场景而生的文档工具
  • GitBook (opens new window)
