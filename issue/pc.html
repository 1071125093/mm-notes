# transform 导致字体不清晰

# 原因

transform 在渲染非整数的 px 时就会出现字体模糊。

# 解决方案

方案一

目标元素 {
  -webkit-font-smoothing: antialiased;
}

方案二

目标元素 {
  transform: perspective(1px);
}
