  2. 多功能输入框，支持文本、截图、图片、文件、文件夹
  3. 自动识别 / 粘贴
  4. 全局快捷键
  5. 云端数据同步
  6. 支持 Windows、Mac、Linux

# QtScrcpy

通过 USB 或 TCP/IP 连接 Android 设备，并进行显示和控制

  1. 不需要 root 权限
  2. 最多支持 16 个安卓设备同时连接
  3. 支持自定义按键映射
  4. 可同时控制所有连接设备

# 图床工具 PicGo

一个用于快速上传图片并获取图片 URL 链接的工具

  1. 支持 GitHub、七牛云、腾讯云、又拍云、阿里云 OSS、SM.MS V2、Imgur
  2. 支持拖拽图片上传
  3. 支持快捷键上传剪贴板里第一张图片
  4. 支持右键图片文件通过菜单上传
  5. 支持自定义复制到剪贴板的链接格式、
  6. 丰富的插件系统open in new window

# Sourcetree

  1. 界面简洁、美观
  2. 简化 Git 日常操作，对新人友好易上手
  3. 支持 Git 工作流，规范开发流程
  4. 支持 Git 和 Mercurial 两种 VCS
  5. 支持自定义脚本执行
