  • 二维码生成器 (opens new window)为当前网页链接或任意内容生成二维码

  • uBlock Origin (opens new window)广告拦截工具

  • 过滤广告 为 Youtube™ (opens new window)移除 Youtube 中的视频广告。

  • The Great Suspender (opens new window)冻结暂时用不到的标签页，以释放系统资源

  • 彩云小译 (opens new window)全文翻译(中英对照)

  • 划词翻译 (opens new window)划词翻译

  • 沙拉查词 (opens new window)Saladict 沙拉查词是一款专业划词翻译扩展，为交叉阅读而生。大量权威词典涵盖中英日韩法德西语，支持复杂的划词操作、网页翻译、生词本与 PDF 浏览。

  • Video Speed Controller (opens new window)视频倍速播放，支持使用快捷键加快，减慢，推进和回放任何 HTML5 视频

  • medium-unlimited (opens new window) 免费读取 medium.com 文章

# 开发常用

  • Vue.js devtools (opens new window)用于调试 Vue.js 应用程序

  • React Developer Tools (opens new window)用于调试 React 应用程序

  • Redux DevTools (opens new window)用于调试应用程序 Redux 状态更改
